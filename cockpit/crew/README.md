# crew/

Optional crew photos for the bridge header: one square-ish JPG per crew
member, named by the `id` field in `config.json` (e.g. `captain.jpg`,
`engineer.jpg`, `science.jpg`). If a photo is missing the display falls back
to that crew member's emoji — which is exactly what a public/demo install
should do. Photos are gitignored: never commit your kids' faces.
