# videos/

Optional viewscreen clips, one per control id (e.g. `gear_up.mp4`,
`master_engine.mp4`). When a control fires and a matching clip exists, the
main viewscreen plays it instead of a still image. We generated ours with an
AI video model (VEO) from prompts like "cinematic shuttle landing gear
retracting as the craft climbs to orbit, view from a chase camera".
Gitignored — bring your own.
